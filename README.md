# Wedding Planner Frontend - Luxury Design

## Where to put this

Put this folder into your repository next to `server`:

```txt
realizace-backendu-JanHousa/
  server/
  client/
```

## Start backend

```bash
cd server
npm install
npm start
```

Backend must run on:

```txt
http://localhost:8888
```

## Start frontend

Open another terminal:

```bash
cd client
npm install
npm run dev
```

Open:

```txt
http://localhost:5173
```
